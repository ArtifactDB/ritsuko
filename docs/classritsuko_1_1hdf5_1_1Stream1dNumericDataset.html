Detailed Description

template<typename Type_>
class ritsuko::hdf5::Stream1dNumericDataset< Type_ >

Stream a numeric 1D HDF5 dataset into memory.

Template Parameters
Type_Type to represent the data in memory.

This streams in a HDF5 dataset in contiguous blocks, using block sizes defined by pick_1d_block_size(). Callers can then extract one value at a time or they can acquire the entire block.
