Detailed Description

Iterate through an N-dimensional dataset by block.

This iterates through an N-dimensional dataset in a blockwise fashion, constructing H5::DataSpace objects to enable callers to easily read the dataset contents at each block. Block sizes are typically determined from dataset chunking via pick_nd_block_size(), which ensures efficient access of entire chunks at each step.
