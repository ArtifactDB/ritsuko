template<class Function_ >
void ritsuko::hdf5::iterate_1d_blocks ( hsize_t  full_length,
hsize_t  block_size,
Function_  fun 
)

Iterate through a 1-dimensional dataset via contiguous blocks.

Parameters
full_lengthLength of the dataset, usually obtained from get_1d_length().
block_sizeSize of the blocks, usually calculated by pick_1d_block_size().
funFunction that accepts (hsize_t start, hsize_t len, H5::DataSpace& memspace, H5::DataSpace& dataspace) and is called on each block. In each call, the block contains elements from [start, start + len). dataspace is configured to extract that block from the dataset, while memspace is configured to deposit the block contents in a buffer from [0, len). It can be assumed that consecutive calls to fun will operate on consecutive contiguous blocks.

Pick a block size to use for 1-dimensional iteration over a dataset. For compressed datasets, this aims to be the smallest multiple of the chunk size that fits into a buffer.

Parameters

Returns
The block size (in terms of the number of elements).
